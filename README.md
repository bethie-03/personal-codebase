# LABEL TOOL TRADING 

## Các bước triển khai

### 1. Clone repository

```
git clone https://github.com/bethie-03/personal-codebase.git
cd personal-codebase
```

### 2. Chuyển sang nhánh label_tool
```
git checkout label_tool
```
### 3. Tạo thư mục cần thiết
```
mkdir image
mkdir reasoning
```
### 4. Chạy Docker Compose
```
docker compose up
```
### 5. Truy cập ứng dụng
```
http://localhost:7860
```
