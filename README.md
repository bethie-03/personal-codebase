# LABEL TOOL TRADING 

## Các bước triển khai

### 1. Clone repository

```
git clone --branch label_tool --single-branch https://github.com/bethie-03/personal-codebase.git
cd personal-codebase
```

### 2. Tạo thư mục cần thiết
```
mkdir image
mkdir reasoning
```
### 3. Chạy Docker Compose
```
docker compose up
```
### 4. Truy cập ứng dụng
```
http://localhost:7860
```
